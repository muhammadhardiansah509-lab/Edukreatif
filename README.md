# Edukreatif

Website untuk membantu guru menciptakan pembelajaran kreatif dan membuat murid lebih bersemangat belajar!

## Fitur
- Materi interaktif
- Quiz dan game edukasi
- Tools generator untuk guru

## Tech Stack
- HTML, CSS, JavaScript

## Cara Menjalankan Lokal
1. Clone repo: `git clone https://github.com/muhammadhardiansah509-lab/Edukreatif.git`
2. Buka index.html di browser.
